# 🌸 Bloom & Petals - E-Commerce UI

A beautifully designed, fully responsive modern frontend web application for a flower shop and delivery service. This project focuses on delivering a seamless User Interface (UI) and User Experience (UX) for browsing floral arrangements.

🚀 **[View Live Demo Here](https://bloom-and-petals-react-mf8m.vercel.app/)**

## ✨ Features
* **Modern Hero Section:** Eye-catching landing area with clear call-to-actions.
* **Product Categories:** Beautifully organized categories for easy navigation.
* **Featured Shop:** A dynamic shop section displaying top-selling flower arrangements.
* **Customer Testimonials:** Trust-building review section.
* **Contact Form:** Interactive form for user inquiries.
* **Fully Responsive:** Perfectly optimized for Desktop, Tablet, and Mobile screens.

## 🛠️ Tech Stack
* **Framework:** React.js
* **Build Tool:** Vite (for lightning-fast development)
* **Styling:** Custom CSS
* **Deployment:** Vercel

## 💻 Getting Started (Run Locally)

If you want to run this project on your local machine, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/Udara-Dilshan/Bloom-and-Petals-REACT.git](https://github.com/Udara-Dilshan/Bloom-and-Petals-REACT.git)
   ```
   
2. **Navigate to the project directory:**
   ```bash
   cd Bloom-and-Petals-REACT
   ```
   
3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Start the development server:**
   ```bash
   npm run dev
   ```
